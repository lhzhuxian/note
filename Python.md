### virtual environment management

```bash
// 创建venv
python -m venv snow-venv 
// 激活虚拟环境
source snow-venv/bin/activate
//使用 pip 管理依赖
pip install flask
更新依赖：
pip install --upgrade flask
pip install -U flask
卸载依赖：
pip uninstall flask
//手动生成依赖列表
pip freeze > requirements.txt
// 安装依赖
pip install -r requirements.txt
```
